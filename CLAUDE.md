# Claude Project Instructions

Read and follow `AGENTS.md`.

Most important rule: every git commit must follow Conventional Commits.

Before committing, always show the exact commit message.

Use format:

<type>[optional scope]: <description>

Examples:
- feat(time): add current time endpoint
- test(time): cover current time endpoint
- docs(readme): document available API endpoints
- ci(gitlab): add basic pipeline
- refactor(time): extract current time provider service
